# Framework — SERP Pattern Analysis

## Wozu

Bevor du eine Zeile schreibst: Was Google bereits ausspielt, ist die einzige verlässliche Aussage darüber, was Google für ein Keyword als "relevant" einstuft. SERP Pattern Analysis macht das systematisch.

Output: Du weißt **welches Format** Google erwartet, **welche Lücke** alle Top-Ergebnisse offen lassen, und **welchen Angle** du spielen kannst, um aufzufallen ohne das erwartete Format zu brechen.

## Die 3-Schichten-Struktur

### Schicht 1: Pattern (was alle tun)

Was machen die Top-5 Ergebnisse alle gleich?

- Format (Listicle / Guide / Comparison / Glossar / Hybrid)
- Länge (Kurz <800 / Mid 800-2000 / Lang 2000+)
- Header-Struktur (welche H2s tauchen bei 4+ von 5 auf?)
- Tonalität (akademisch / casual / werblich / aktivistisch)
- Quellen (Eigenexpertise / Studien / User-Quotes)
- CTA-Architektur (sanft / hart / keine)

**Regel:** Die Pattern existiert weil Google sie belohnt. Bricht du das Pattern komplett, rankst du nicht. Bricht du es in einer Dimension, hast du eine Chance.

### Schicht 2: Weakness (was alle gleich schlecht machen)

Was ist bei 4+ Top-5 Ergebnissen das gleiche Defizit?

Häufige Patterns:
- Generische Eingangs-Absätze ("In der heutigen Zeit...")
- Listen ohne Substanz ("Top 10" mit 50-Wort-Beschreibungen)
- Marketing-Sprache ohne Beweis
- Veraltete Daten (älter als 18 Monate)
- Fehlende echte Beispiele
- Keine Persona-Empathie (Texte klingen wie an niemanden gerichtet)
- Schwache visuelle Architektur
- Reiner SEO-Content ohne Standpunkt

**Regel:** Die Weakness ist deine Eintrittstür. Wenn alle es gleich schlecht machen, ist "es gut machen" Differenzierung.

### Schicht 3: Opportunity (dein Hebel)

Aus Pattern + Weakness extrahierst du den Opportunity-Angle. Das ist ein konkreter Satz, kein Konzept.

**Schwacher Angle:** "Wir machen es besser."

**Starker Angle:** "Wir liefern statt 10 Top-Items 3 mit echten Tasting-Notes, Preis-Pro-Gramm-Vergleich und einem Test-Setup, das Leser nachstellen können."

Der Angle muss:
1. Im Format-Korridor bleiben (Pattern respektieren)
2. Die Weakness adressieren
3. Konkret-überprüfbar sein (nicht "besser", sondern "statt X jetzt Y")
4. Für deine Brand exklusiv funktionieren (Substitution-Test)

## Die Anwendungs-Sequenz

```
1. SERP-Snapshot machen (manuell oder Tool)
   → Top 10 URLs, Titles, Meta-Descriptions, Position-0-Inhalt, PAA, Related Searches

2. Pro Top-5: Format-Klassifikation
   → Format / Länge / Header-Struktur / Tonalität / Quellen / CTAs

3. Pattern-Extraktion (was machen 4+ gleich?)

4. Weakness-Extraktion (was fehlt bei 4+?)

5. Opportunity-Angle formulieren (1 Satz, konkret, überprüfbar)

6. Eigenes Format ableiten (Pattern + Angle = dein Brief)
```

## Qualitäts-Test für deinen Opportunity-Angle

Stell deinem Angle 4 Fragen:

**1. Konkret-Test:** Steht im Angle eine Größe, eine Zahl, oder ein Beweisstück, das messbar ist? ("3 statt 10", "Tasting-Notes statt Marketing-Floskeln")

**2. Substitutions-Test:** Wenn du den Wettbewerber einsetzt — funktioniert der Angle dann auch für ihn? Wenn ja: zu generisch.

**3. Kosten-Test:** Kannst du den Angle wirklich umsetzen? Wenn er "echte Tasting-Notes" verspricht, hast du die Tasting-Notes? Wenn nicht: Angle ändern oder Daten beschaffen.

**4. Persona-Test:** Würde deine Persona den Angle attraktiv finden? Oder ist es ein Angle, der Marketers gefällt aber Personas kalt lässt?

## Häufige Fehler

**Fehler 1: Pattern komplett brechen**
Brand schreibt einen Listicle als Essay-Form, weil "Listicles sind langweilig". Resultat: Google erkennt das nicht als zur Search Intent passend, rankt nicht.

**Fix:** Pattern in EINER Dimension brechen, nicht in allen. Wenn alle Listicles 10 Items haben, mach 5 mit doppelter Tiefe. Format bleibt, Substanz wird neu.

**Fehler 2: Pattern blind kopieren**
Brand baut eine 1:1-Kopie der Top-1 Struktur. Google sieht keinen Mehrwert, rankt nicht. Auch nicht bei perfektem Schema.

**Fix:** Pattern + EIGENER Angle. Pattern allein reicht nicht für Top-3.

**Fehler 3: Weakness nicht echt prüfen**
Brand denkt sich eine Weakness aus ohne SERP wirklich zu lesen. Schreibt Content der eine ausgedachte Lücke schließt.

**Fix:** Mindestens 5 Top-Ergebnisse vollständig lesen (nicht skimmen). Notiere wörtlich was fehlt. Dann clusterst du.

**Fehler 4: Opportunity-Angle ist "besser" oder "umfassender"**
Diese Worte sind keine Angles. Sie sind Marketing-Floskeln.

**Fix:** Schreibe den Angle als WENN-DANN-Satz: "WENN ich diesen Artikel lese, DANN bekomme ich [konkret X], was ich woanders nicht bekomme."

## Wie tief gehen?

Faustregel:

- **Hochkompetitive Keywords (Suchvolumen 5K+):** Vollanalyse, alle 10 Top-Ergebnisse, mindestens 2h Arbeit
- **Mittlere Keywords (1K-5K):** Top 5 reicht, 45-60 Min
- **Long-Tail (unter 1K):** Top 3 reicht, 20-30 Min — bei Long-Tail ist die Pattern oft schwach (= mehr kreative Freiheit)

## Output-Form (was du nach SERP-Analyse haben solltest)

Pro Keyword:

- Pattern (1 Absatz)
- Weakness (1 Absatz)
- Opportunity-Angle (1 Satz)
- Format-Vorgabe (Format-Typ + Länge + grobe H2-Struktur)
- Realismus-Einschätzung (Top 3 möglich? Top 10? oder unrealistisch?)

Diese 5 Punkte sind der Input für deinen Content-Brief. Ohne sie schreibst du blind.
