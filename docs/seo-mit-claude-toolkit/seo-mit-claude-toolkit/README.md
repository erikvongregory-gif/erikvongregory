## So gehst du vor (3 Wege)

**Weg 1 — End-to-End:** Lies `01-WORKFLOW-MAP.md`, dann arbeite die 8 Phasen in Reihenfolge ab. ~1-2 Tage Strategie + 3-5 Tage Content/Tech.

**Weg 2 — Single Phase:** Du hast schon SEO und willst eine Phase fixen → such dir den passenden Prompt aus `02-PROMPTS/`.

**Weg 3 — Framework First:** Du willst die Denke verstehen bevor du handelst → lies zuerst `03-FRAMEWORKS/`.

## Bevor du den ersten Prompt laufen lässt

Lies `04-CHECKLISTS/prompt-input-quality.md`. Der Prompt macht nicht die Qualität — dein Input macht sie.

Ohne Brand-Kontext, Persona, Constraints liefert Claude generischen Content. Mit dem richtigen Input liefert er Arbeit, für die Agenturen fünfstellig schreiben.

## Was das Toolkit nicht ist

Kein Link-Building, kein Outreach, kein Local SEO, kein Ersatz für GSC/Ahrefs. Es ist Arbeit an On-Page, Content-Strategie, Technical SEO, Schema und Meta — also 80% dessen, was du mit AI hochwertig machen kannst.
